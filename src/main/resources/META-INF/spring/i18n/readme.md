### Readme Nomenclature .properties 

Readme sur l'appellation des éléments dans les .properties

### Utilsation des trigrames 


* **pdt** = Produit
* **pan** = Panier
* **cmd** = Commande
* **usr** = Utilisateur
* **glb** = Global
* **mlg** = mention legal
* **con** = contact

### Exemple 
la nomenclature a suivre et la suivante :

**nom du trigramme + numero ecran**.**nomDeLaCategorie**.**nom de l'element si nécessaire**

### Global 

exemple : 

-glb.header.titre

-glb.footer.lien.con

-glb.footer.lien.mlg


### Produit

exemple **PDT_00 :**

**titre de page**

* pdt00.titre

 **erreurs**
 
* pdt00.erreur.nomErreur

 **succès**
 
* pdt00.validation.nomValidation

 **textes**

* pdt00.label.nomdulabel
